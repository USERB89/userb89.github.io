3 урок. Виды глагола — أَنْوَاعُ الْفِعْل

Глаголы в арабском языке делятся на три вида:

  1. Глагол прошедшего времени (الفِعْل الماضِي) — Например: قَرَأَ «читал»
  2. Глагол настоящего-будущего времени (الفِعْل المُضارِع) — Например: يَقْرَأُ «читает»
  3. Глагол повелительного наклонения (فِعْل الأَمْر) — Например: اِقْرَأْ «читай»

Внимательно посмотрите на следующие предложения:

Мухаммад читал эту книгу .قَرَأَ محمدٌ هذا الكتابَ
Мухаммад читает эту книгу .يَقْرَأُ محمدٌ هذا الكتابَ
Читай эту книгу, эй Мухаммад .اِقْرَأْ هذا الكتابَ، يا محمدُ

Теперь рассмотрим ещё примеры:

Ахмад уснул рано .نامَ أحمدُ مُبَكِّرًا
Ахмад рано засыпает .يَنامُ أحمدُ مُبَكِّرًا
Ложись спать рано, Ахмад .نَمْ مُبَكِّرًا، يا أحمدُ
Махмуд ел мало .محمودٌ أَكَلَ قليلًا
Махмуд ест мало .محمودٌ يَأْكُلُ قليلًا
Ешь мало, Махмуд .كُلْ قليلًا، يا محمودُ
Ибрахим учил правило .دَرَسَ إبراهيمُ القاعدةَ
Ибрахим учит правило .يَدْرُسُ إبراهيمُ القاعدةَ
Учи правило, Ибрахим .اُدْرُسْ القاعدةَ، يا إبراهيمُ

Список глаголов

Выпишите и выучите:
  • نَظَرَ / يَنْظُرُ / اُنْظُرْ — «смотреть»
  • بَدَأَ / يَبْدَأُ / اِبْدَأْ — «начинать»
  • اِنْتَظَرَ / يَنْتَظِرُ / اِنْتَظِرْ — «ждать»
  • بَاعَ / يَبِيعُ / بِعْ — «продавать»
  • اِشْتَرَى / يَشْتَرِي / اِشْتَرِ — «покупать»
  • حَفِظَ / يَحْفَظُ / اِحْفَظْ — «учить, хранить»
  • اِسْتَيْقَظَ / يَسْتَيْقِظُ / اِسْتَيْقِظْ — «просыпаться»
  • أَحَبَّ / يُحِبُّ / أَحِبَّ — «любить»
  • نَزَلَ / يَنْزِلُ / اِنْزِلْ — «спускаться»
  • أَعْطَى / يُعْطِي / أَعْطِ — «давать»
  • سَكَنَ / يَسْكُنُ / اُسْكُنْ — «жить»
  • صَلَّى / يُصَلِّي / صَلِّ — «совершать намаз»
  • سَافَرَ / يُسَافِرُ / سَافِرْ — «путешествовать»
  • فَهِمَ / يَفْهَمُ / اِفْهَمْ — «понимать»

УПРАЖНЕНИЯ

1. В пустые места вставьте подходящие глаголы настоящее — будущего времени из предыдущего списка:

.المسلمُ يُصَلِّي خمسَ مَرَّاتٍ كُلَّ يَوْمٍ

.هو يَسْكُنُ في شَقَّةٍ نظيفةٍ

.هو يَسْتَيْقِظُ عندَ الفجرِ

.يُحِبُّ الإنسانُ أولادَه

2. В пустые места вставьте подходящие глаголы прошедшего времени:

.سَافَرَ أبي إلى مكةَ للعُمْرةِ

.نَزَلَ المَطَرُ مِن السَّماءِ

بَاعَ أخي درَّاجتَه القَدِيمَةَ، و اِشْتَرَى درَّاجةً جَدِيدَةً

.حَفِظَ الطفلُ بعضَ سُوَرِ القرآنِ الكريمِ
